# Vetro Restaurant & Lounge — Website

Marketing site for Vetro by Russo's on the Bay, a waterfront Italian restaurant, lounge, and catering venue in Howard Beach, NY.

## Structure
- `index.html` — landing page (overview of restaurant, lounge, and the five event spaces)
- `vetro-logo.webp` — logo (keep in the same folder as the HTML)

## Status
Landing page built. Restaurant page (menu + drinks + Blue Lounge) and event pages (Rooftop, Murano Room, Wine Cellar) to come.

Image slots are labeled placeholders awaiting real photography.
